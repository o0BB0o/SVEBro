# SVE Bro

## 简介
使用你心爱的主站者吧大概。可以配合print使用（广告位出租）。

<img src="https://github.com/o0BB0o/SVEBro/blob/main/readme_res/Screenshot_20241020_152856.png" alt="demo_img" width="500"/>

## 使用方法
按照模板（模板.zip）将所需文件打包成zip，进入app后输入相关内容后将生成该主站者。
